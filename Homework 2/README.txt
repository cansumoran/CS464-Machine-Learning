You can run the script driver.txt with the following command from the terminal:
. driver.txt
The driver script will run the code for question 1, question 2 and question 3 in the given order.

Before running the script, you should create a virtual environment with the used libraries.
The requirements.txt includes which libraries and which versions should be used.
You can install the libraries in requirements.txt with the following command from the terminal:
pip install -r requirements.txt

Before running the script, place the datasets to be used for each code into a folder named "data".
